# Job Description Keyword Analyzer (Python)

A simple Python command-line tool that analyzes job descriptions and extracts
frequently used keywords and skill phrases.

## Features
- Cleans and tokenizes job description text
- Removes common stop words
- Identifies important skill-related keywords
- Exports keyword frequencies to a CSV file

## How to Run
1. Make sure Python 3 is installed
2. Run the script:
   ```bash
   python3 keyword_analyzer.py
END
